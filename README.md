# Application de gestion et suivi des stagiaires

## Description
Cette application web permet de gérer les stagiaires, leurs tâches,
les notifications et le suivi des activités.

## Fonctionnalités
- Authentification (Admin / Stagiaire)
- Gestion des tâches
- Notifications en temps réel
- Commentaires et réponses
- Suivi des activités

## Technologies utilisées
- PHP
- MySQL
- HTML / CSS / JavaScript
- Bootstrap
- AJAX

## Installation
1. Cloner le projet
2. Importer la base de données MySQL
3. Configurer le fichier `config.php`
4. Lancer le projet sur un serveur local (XAMPP)
